These commands provide metadata about the installed bino CLI.
They are useful for debugging, support, and license compliance.

## bino version

```bash
bino version
```

- Prints the bino CLI version.
- Use it in issue reports and CI logs.

## bino about

```bash
bino about
```

- Prints product and about information.
- Includes a concise list of direct dependencies with license identifiers and URLs.

You can copy this output into internal documentation when tracking third-party dependencies.

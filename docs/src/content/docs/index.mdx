## Quick Install

```bash
/bin/bash -c "$(curl -fsSL https://github.com/bino-bi/bino-cli-releases/releases/latest/download/install.sh)"
```

## Why Bino?

<CardGrid stagger>
  <Card title="Declarative & Version Controlled" icon="document">
    Define your entire reporting pipeline in YAML and SQL. Treat your reports
    like code—commit, diff, and review changes with Git.
  </Card>
  <Card title="Data Agnostic" icon="layers">
    Connect to any data source supported by DuckDB. Query Postgres, MySQL,
    Parquet, CSV, and JSON directly from your report definition.
  </Card>
  <Card title="Pixel-Perfect PDFs" icon="pencil">
    Design layouts with precision. Control pagination, headers, footers, and
    typography to generate professional-grade PDF documents.
  </Card>
  <Card title="Developer Experience" icon="terminal">
    Built for developers. Enjoy a fast CLI, instant local preview, hot
    reloading, and TypeScript-like validation for your report schemas.
  </Card>
</CardGrid>
